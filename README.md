# ibm-article-recommendations

This repository contains a recommendation system project analyzing user interactions with articles on the IBM Watson Studio platform.

## Files Included
- `Recommendations_with_IBM.ipynb` – Main project notebook (run top to bottom)
- `user-item-interactions.csv` – User-article interaction data
- `articles_community.csv` – Article content metadata

## How to Run
1. Clone or download the repository
2. Open the notebook in Jupyter
3. Run all cells in order

## Recommendation Techniques
- Rank-Based Recommendations
- User-User Collaborative Filtering
- Content-Based Recommendations
- Matrix Factorization (SVD)
